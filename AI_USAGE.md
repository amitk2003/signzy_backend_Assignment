# AI Usage Disclosure

In accordance with the assignment requirements, this document outlines the usage of AI tools during the development of this project.

**Tool Used:** Google Gemini 
**Role:** AI Coding Assistant

## How AI was used:

1. **Architecture Planning**: I used the AI to help structure the initial project layout (separating routes, strategies, services, and store) and validate the in-memory data store approach using Python threading locks.
2. **Boilerplate Code**: The AI generated boilerplate for the Flask routes and standard JSON validation logic.
3. **Strategy Implementations**: I provided the logic requirements for the 8 routing strategies, and the AI helped write the Python implementation for the weighted and round-robin strategies.
4. **Simulator Logic**: The AI helped design the `vendor_caller.py` simulator to generate realistic random latencies and failures, ensuring the failover logic could be properly tested.
5. **Documentation Formatting**: The AI assisted in formatting the markdown files (README, ARCHITECTURE) for clarity and readability.

The core logic, decision trees, edge case handling, and overall system design were directed by human intent to fulfill the specific assignment requirements.
